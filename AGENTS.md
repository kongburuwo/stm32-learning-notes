# AGENTS.md

This repository contains personal study notes for C language and STM32 embedded development.

## Key guidance for AI agents

- Treat this repo as a learning notebook, not a production application.
- Prefer explanations, summaries, and study guidance over writing or completing code.
- Do not provide automatic code completion unless the user explicitly requests an example or a complete implementation.
- If the user asks about code, explain concepts, point out where existing examples are, or show only short illustrative snippets.
- Avoid making assumptions about missing code; ask for clarification if the request is unclear.

## Repository structure

- `01-C/` and `01-C语言基础/`: C language fundamentals, exercises, and notes.
- `02-电路基础/`: circuit concepts such as pull-up/pull-down and transistor basics.
- `03-STM32系统入门/` and `STM32/`: STM32 peripheral and interrupt notes, plus project template documentation.
- `汇编入门/`: assembly language introduction.

## Useful behavior

- When asked to help with a learning note, reference the relevant directory and explain the underlying principle.
- When asked about STM32 topics, focus on embedded concepts and peripheral usage rather than generating full firmware code.
- Keep answers concise and educational.

## Notes

- There is no centralized build or test configuration in this repository.
- Use existing markdown notes as source material when possible.
