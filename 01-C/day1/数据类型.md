## bit和byte的关系
- bit（比特）：计算机最小存储单元，只能是0或1，相当于一个电子开关
- byte（字节）：基本计量单位，1 byte=8 bit，能表示256种状态（2⁸）
- 示例：int age=20;在内存中占用00000000 00000000 00000000 00010100的二进制形式

## 整型数据类型
| 数据类型   | 修饰符   | 占用空间                                |
| :--------- | :------- | :-------------------------------------- |
| short      | signed   | 2字节                                   |
|            | unsigned | 2字节                                   |
| int        | signed   | 4字节                                   
|            | unsigned | 4字节                                   |
| long       | signed   | Windows占4字节，Linux占8字节           | 
|            | unsigned | Windows占4字节，Linux占8字节           | 
| long long  | signed   | 8字节                                   |
|            | unsigned | 8字节                                   |


## 格式符匹配  
| 数据类型   | 修饰符   | 输出格式符 |
| :--------- | :------- | :--------- |
| short      | signed   | %hd        |
|            | unsigned | %hu        |
| int        | signed   | %d         |
|            | unsigned | %u         |
| long       | signed   | %ld        |
|            | unsigned | %lu        |
| long long  | signed   | %lld       |
|            | unsigned | %llu       |

## 宏定义
- #define宏：定义的常量全局有效，不受作用域限制；

## const
- 声明只读变量（实质为常量），遵循变量作用域规则；
