## 什么是作用域
- 定义：变量在程序中能被访问到的有效范围，超出这个范围则变量不可见或不可用。

## 作用域分类
1. 局部变量
- 生活类比：像教室里的黑板，只能在当前教室使用。  
- 定义位置：函数内部或代码块（如{}）内定义的变量。  
- 作用域范围：  
  - 仅限定义它的函数或代码块内部。  
  - 函数参数（如int a）也是局部变量。  
- 生命周期：  
  - 函数调用时创建，函数结束时销毁

2. 全局变量
- 生活类比：像校园最高的那栋楼，全校师生都能看到。  
- 定义位置：在所有函数外部定义的变量。  
- 作用域范围：  
  - 整个程序（从定义位置开始到文件结束）。  
  - 其他文件通过extern声明后也可使用。  
- 生命周期：  
  - 程序运行时创建，程序结束时销毁。
3. 常见错误
- 在函数外访问局部变量
```c
    void func() { int a = 5; }
    int main() { printf("%d", a); }  // 编译错误：a未定义 
```

- 全局变量与局部变量同名
```c
    int num = 100;  // 全局变量 
    void func() {
        int num = 200;  // 局部变量 
        printf("%d", num);  // 输出200（就近原则）
}
```

- 未初始化的局部变量
```c
    void test() {
        int x;  // 未初始化，值为随机垃圾数据 
        printf("%d", x);  // 可能输出奇怪数值或崩溃 
    }
```
