//冒泡：n-1轮，每轮比较相邻的两个数，如果前一个数比后一个数大，就交换它们的位置。每轮结束后，最大的数就会被放到正确的位置上。
void bubbleSort(int a[], int n)
{
    for (int i = 0; i < n - 1; i++)
    {
        for (int j = 0; j < n - i - 1; j++)
        {
            if (a[j] > a[j + 1])
            {
                int temp = a[j];
                a[j] = a[j + 1];
            }
        }
    }
}
