## 中断的概念
- 单片机正在正常跑主程序，突然来了一件急事，暂时停下主程序，先去处理急事，处理完再回来接着跑原来的程序。

```c
int main(void)
{
    while (1)
    {
        //主程序做的事情
    }

    void EXTIO_IRQHandler(void)
    {
        //中断做的事情
    }
}
```
## 中断的意义
- 不用 CPU 主动不停查询，事件来了自动通知、立刻处理，既省资源、又响应快、还能同时干多件事

## 中断的优先级
- 中断具有优先级，两个中断同时触发，优先响应高优先级的中断

## 中断嵌套
- 正在处理一个中断的时候，又来了一个优先级更高的中断，插队进来先执行，这就是中断嵌套
