# EXTI
- EXTI：(External Interrupt / Event Controller)外部中断/事件控制器
- 作用：让单片机可以响应外部引脚的电平变化（上升沿 / 下降沿），触发中断或事件。
- 中断：打断CPU主程序，跳转执行中断服务函数（喊 CPU 过来干活）
- 事件：只产生触发信号，不打断 CPU、不进中断（硬件之间自己干活，不打扰 CPU）
- 流程：GPIO → AFIO（选线） → EXTI（触发） → NVIC（放行） → 中断服务函数

## EXTI 寄存器描述
- 共5个核心寄存器，均32位，低19位有效(bit0~bit18)，对应 EXTI0 ~ EXTI18

### EXTI寄存器及相关函数
1. 中断屏蔽寄存器(EXTI_IMR)
    - 中断总开关
    - 位 = 1：允许该线产生中断请求
    - 位 = 0：屏蔽中断，禁止发往 NVIC
- 事件屏蔽寄存器(EXTI_EMR)
    - 硬件事件通路开关，不走 NVIC
    - 位 = 1：允许输出事件信号
    - 位 = 0：屏蔽事件
- 上升沿触发选择寄存器(EXTI_RTSR)
    - 位 = 1：开启上升沿触发（低→高）
    - 位 = 0：关闭上升沿
- 下降沿触发选择寄存器(EXTI_FTSR)
    - 位 = 1：开启下降沿触发（高→低）
    - 位 = 0：关闭下降沿
