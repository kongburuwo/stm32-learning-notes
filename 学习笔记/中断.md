# 中断
## 中断简介
- *中断*：中断是一种特殊信号，当发生某些事件时，CPU会自动响应并执行相应的处理程序。
- 中断处理程序：中断处理程序是处理中断的函数，当发生中断时，CPU会自动调用该函数。
- 中断发生时，CPU会保存当前执行位置，并跳转到中断处理程序，执行中断处理程序，处理完成后，CPU会返回到保存的位置继续执行。

## NVIC概念和工作原理
### NVIC概念
- NVIC（Nested Vectored Interrupt Controller）：嵌套向量中断控制器，是STM32F1xx系列芯片 interrupt controller 的简称。是集成在 Cortex-M3 内核里的专用硬件，所有外设中断、内核异常全部统一交给 NVIC 管理。
- *作用*：接收芯片全部中断请求，做优先级仲裁、中断开关、中断嵌套控制，通知 CPU 跳转执行中断服务程序。
- *Nested 嵌套*：高抢占优先级中断，可以打断正在运行的低优先级中断，实现多层嵌套；
- *Vector 向量*：中断服务程序地址存放在向量表中，通过读取中断向量表，CPU 跳转到相应的中断处理程序。
### NVIC 整体信号流向
- 外设硬件（EXTI/TIM/USART）产生*中断请求* → 外设*中断使能*打开 → 信号送入*NVIC* →
NVIC 对比当前正在执行中断的优先级 → *判定*是否抢占 →
通知内核暂停主程序 / 低优先级中断，跳转到对应*中断服务函数*。
### 两大优先级：抢占优先级 + 子优先级
1. 抢占优先级（主优先级）：
- 决定能不能打断别人，支持中断嵌套；
- 数字越小优先级越高；
- 高抢占优先级中断到来时，若 CPU 正在运行低抢占中断，立刻保存现场，切入新中断。
2. 子优先级（响应优先级）：
- 只在同一时刻多个中断同时触发时生效；
- 不能打断正在执行的中断，无嵌套能力；
- 多个中断一起进来，子优先级数字小的先执行。
### 优先级分组（全局唯一，整个程序只能配置 1 次）
- 通过 HAL_NVIC_SetPriorityGrouping() 设置，用来分配抢占位、子优先级位：
    示例：GROUP2，抢占 0~3，子优先级 0~3；抢占 0 可以抢占抢占 1/2/3 的中断。
    ```c
    // 1. 全局设置优先级分组（程序开头只调用一次）
    HAL_NVIC_SetPriorityGrouping(NVIC_PRIORITYGROUP_2);
    ```
### NVIC代码配置流程
    ```c
    // 1. 全局设置优先级分组（程序开头只调用一次）
    HAL_NVIC_SetPriorityGrouping(NVIC_PRIORITYGROUP_2);
    // 2. 配置中断源优先级
    HAL_NVIC_SetPriority(EXTI0_IRQn, 2, 0);
    // 3. 启动中断源
    HAL_NVIC_EnableIRQ(EXTI0_IRQn);
    ```

## EXTI概念和工作原理
### EXTI概念
- EXTI（External Interrupt）：外设中断，是 STM32F1xx 系列芯片 interrupt controller 的简称。外设中断是外设产生中断信号，并通知 CPU 跳转到中断处理程序。
- *作用*：接收外设中断信号，做中断开关、中断嵌套控制，通知 CPU 跳转执行中断服务程序。
- *中断源*：外设产生中断信号，如按键按下、定时器溢出、串口接收完成、DMA传输完成等。
### EXTI信号流程
1. 开启时钟
    - GPIO 端口时钟（如 GPIOA）__HAL_RCC_GPIOA_CLK_ENABLE();
    - AFIO 复用 IO 时钟 __HAL_RCC_AFIO_CLK_ENABLE();
2. GPIO 初始化
    - 引脚设为中断模式：GPIO_MODE_IT_RISING / IT_FALLING / IT_RISING_FALLING
3. AFIO 配置
    - 选择该引脚对应 EXTI 线路使用哪个端口（PA/PB/PC…）
4. EXTI 配置
    - 触发边沿选择（上升 / 下降 / 双边沿）
    - 开启 EXTI 对应线路的中断使能（允许向 NVIC 发请求）
5. 外部电平跳变 → EXTI 硬件自动置起 PR 挂起标志位（外设中断标志）
6. 中断请求送入 NVIC
7. NVIC 配置优先级、使能中断通道
8. 触发中断，进入中断服务函数
9. 在中断里手动清零 EXTI->PR 标志位，否则反复进中断
### EXTI 核心寄存器
1. *AFIO_EXTICR1~4*
- AFIO_EXTICR1~4 寄存器用于配置 EXTI 线路所使用的 GPIO 端口。
2. *EXTI_RTSR* 上升沿触发寄存器
- EXTI_RTSR 寄存器用于配置 EXTI 线路触发方式，置1 即上升沿触发。
3. *EXTI_FTSR* 下降沿触发寄存器
- EXTI_FTSR 寄存器用于配置 EXTI 线路触发方式，置1 即下降沿触发。
4. *EXTI_PR* 挂起 / 中断标志寄存器
- EXTI_PR 寄存器用于清除中断标志位，置1 即清除中断标志位。
### EXTI 代码流程
    ```c
    // 1. 开启时钟
    __HAL_RCC_GPIOA_CLK_ENABLE();
    __HAL_RCC_AFIO_CLK_ENABLE();
    // 2. GPIO 初始化
    GPIO_InitTypeDef gpio_initstruct = {0};
    gpio_initstruct.Pin = GPIO_PIN_0;
    gpio_initstruct.Mode = GPIO_MODE_IT_FALLING; // 下降沿外部中断
    gpio_initstruct.Pull = GPIO_PULLUP;
    HAL_GPIO_Init(GPIOA, &gpio_initstruct); 

    // 3. NVIC 配置中断优先级和使能
    HAL_NVIC_SetPriorityGrouping(NVIC_PRIORITYGROUP_2); // 配置中断分组只用写一次
    HAL_NVIC_SetPriority(EXTI0_IRQn, 2, 0);
    HAL_NVIC_EnableIRQ(EXTI0_IRQn);

    // 4. 中断服务函数
    void EXTI0_IRQHandler(void)
    { 
        HAL_GPIO_EXTI_IRQHandler(GPIO_PIN_0); //自动清PR标志
    }
    void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
    { 
        if(GPIO_Pin == GPIO_PIN_0)
        {
            HAL_GPIO_TogglePin(GPIOA,GPIO_PIN_1);
        }
    }
    ```
