# 继电器
## 简介以及原理
- 继电器是一种电磁控制的隔离式开关器件，实现弱电控制强电、小电流控制大电流，是单片机最常用的功率开关元件。原理是利用电磁感应来实现电路的通断控制。 

## 输出部分
- 常开端 NO：默认断开，继电器吸合后导通。适用于大多数“开关控制”场景。
    - 功能：在继电器未通电（静止）状态下，COM 与 NO 是断开的。
    - 动作：当继电器线圈通电吸合后，COM 与 NO 接通。
    - 应用：最常用的接法。用于实现“通电后开启设备”的逻辑，如点亮灯泡、启动电机。
- 常闭端 NC：默认导通，继电器吸合后断开。适用于“失效保护”或反向逻辑控制。
    - 功能：在继电器未通电（静止）状态下，COM 与 NC 是接通的。
    - 动作：当继电器线圈通电吸合后，COM 与 NC 断开。
    - 应用：用于实现“通电后关闭设备”或“故障安全”逻辑。例如，某些安全系统在断电时需要保持电路连通，或者用于互锁电路。
- 公共端 COM：动触点的公共端，随继电器状态在 NO 和 NC 之间切换。
    - 功能：这是负载电路的公共连接点，通常连接到电源的一端或负载的一端。
    - 特点：它始终与动触点相连，根据继电器状态，要么连接到 NO，要么连接到 NC。

## 相关实验
- 开关继电器实验
```c
//初始化GPIO函数
void alarm_init(void)
{
    GPIO_InitTypeDef gpio_initstruct;
    //打开时钟
    __HAL_RCC_GPIOA_CLK_ENABLE();                           // 使能GPIOB时钟
    
    //调用GPIO初始化函数
    gpio_initstruct.Pin = GPIO_PIN_6;                       // alarm对应的引脚
    gpio_initstruct.Mode = GPIO_MODE_OUTPUT_PP;             // 推挽输出
    gpio_initstruct.Pull = GPIO_PULLUP;                     // 上拉
    gpio_initstruct.Speed = GPIO_SPEED_FREQ_HIGH;           // 高速
    HAL_GPIO_Init(GPIOA, &gpio_initstruct);
    //关闭LED
    alarm_off();
}

void alarm_on(void)
{
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_6, GPIO_PIN_RESET);   // 拉低alarm引脚，点亮alarm
}

void alarm_off(void)
{
    HAL_GPIO_WritePin(GPIOA, GPIO_PIN_6, GPIO_PIN_SET);     // 拉高alarm引脚，熄灭alarm
}

uint8_t alarm_status_get(void)
{
    return (uint8_t)HAL_GPIO_ReadPin(GPIOB, GPIO_PIN_6);
}
```
